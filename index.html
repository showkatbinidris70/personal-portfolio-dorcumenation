File/Folder Description
public The React.js public folder can serve static files, allowing you to reference files such as images, fonts scripts starting from the base URL "/".
src The source folder includes all template source files, such as Pages, Components, Data, CSS, JavaScript, and media files.
node_modules A directory created by npm and a way of tracking each packages you install locally via package.json
build A directory created by npm that tracks all the packages you install locally via the package.json file.
package.json Includes the list of dependencies to install from npm
